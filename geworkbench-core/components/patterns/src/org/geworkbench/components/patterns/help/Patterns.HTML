Patterns

The Patterns component is located in the View Area in the top-right of the application. It is used to display resulting sequences found during a SPLASH pattern discovery discoverySession. This panel is necessary because the pattern discovery panel may not be able to display the discovered sequence in its entirety because of its length. Sequences are added to the Patterns panel by clicking on them in the “Patterns Discovery” panel.

View Results in the Pattern View

1 From the Tool and Selection Area, select Pattern Discovery. Complete the steps outlined in Pattern Discovery help section to find sequences.

2 Click on a single sequence in the Pattern Discovery and the Pattern view is updated to include the selected sequence.

3 To select more than one, CTRL-CLICK to select more values or SHIFT-CLICK to select a range of values and the Pattern view is updated to reflect the selected sequences. Each pattern receives a unique color.

4 Clearing the selected sequences is done by pressing the Clear button on top of the Patterns panel.
